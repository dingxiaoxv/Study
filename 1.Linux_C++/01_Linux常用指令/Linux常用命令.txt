权限   硬链接数  属有者  所属组  文件大小   日期  文件名

drwxr-xr-x文件类型及对应权限 
-：普通文件 
d：目录
rwx：文件所有者权限
r-x：同组人权限
r-x：组外人权限
4：硬链接数

ls命令 迭代显示
1、显示该目录下文件
ls -a
2、显示所有目录+隐藏文件（以.开头的文件）
ls -al
3、全部文件以详细列表显示
ls -R|less
在列出的文档名称后加一符号；例如可执行文档则加“*”,目录则加 “/” ，符号链接文件加“@”
ls -F
详细显示，同时文件大小用k表示
ls -lh

cd命令 切换当前工作目录
1、cd ..  更换目录回到上一级目录
2、cd .   更换到当前目录
3、cd ~   更换目录到当前用户的家目录（或者直接cd）
4、cd /   更换目录到根目录
cd [绝对路径]
cd /etc/vsftpd/  更换到/etc/vsftpd目录

查找文件
在/home目录下查找以.txt结尾的文件名
find /home -name "*.txt"
当前目录及子目录下查找所有以.txt和.pdf结尾的文件
find . -name "*.txt" -o -name "*.pdf"
找出/home下不是以.txt结尾的文件
find /home ! -name "*.txt"
查找所有空文件并将其删除。
find / -size 0 Cexec rm Crf {} \;
{} \;命令结束停止符
find . -size 0 -exec rm -i {} \
